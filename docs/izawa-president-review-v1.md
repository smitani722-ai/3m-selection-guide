# 3M製品セレクションガイド 営業ロジック完成版 v1.0

## 1. 目的

本プロジェクトの目的は、3M製品の選定における営業判断を標準化し、営業担当者が短時間で適切な推奨製品を提示できるようにすることです。

従来は、製品選定が担当者の経験や記憶に依存しやすく、同じ問い合わせでも担当者によって提案製品が変わる可能性がありました。特に、VHB、LSE対応テープ、再剥離テープ、構造用接着剤、Dual Lockなどは、製品ごとの使い分けに営業経験が必要です。

本ツールでは、用途、被着体、温度条件、要求特性、厚み、価格重視の有無などを質問形式で入力すると、営業ルールに基づいて推奨製品、代替候補、データシートを表示します。新人教育、営業品質の平準化、問い合わせ対応のスピード向上を目的とした営業支援ツールです。

## 2. 特徴

### 質問に答えるだけで推奨品を表示

営業担当者は、画面上の質問に順番に答えるだけで、用途に合う3M製品を確認できます。専門知識が不足している担当者でも、一定水準の選定結果に到達しやすい設計です。

### 推奨品だけでなく代替候補も表示

第一推奨品に加えて、代替候補も表示します。例えば、Y4825とGPHシリーズ、メタルボンダーとメタルグリップ、1110とST416Pなど、営業現場で説明が必要になりやすい製品関係を整理しています。

### 営業トーク化された推薦理由

単なるスペック説明ではなく、お客様へ説明しやすい言葉に改善しています。

例:

「粗面やコンクリートでもフォームがなじみやすく、現場で失敗しにくい製品です」

「国内で提案しやすく、納期や供給面の説明がしやすい標準VHBです」

このように、営業担当者がそのまま商談で使える表現を重視しています。

### データシートへの導線

推薦結果画面から、登録済みのデータシートを開けるようにしています。現時点では52製品中27製品に資料を登録済みです。今後、未登録製品の資料整備を進めることで、商談時の資料提示力を高めます。

## 4. 今後の拡張計画

### 優先度A: データシート整備

現時点では52製品中27製品にデータシートを登録済みです。社外向け・商談向けの完成度を高めるため、未登録25製品の資料整備が最優先です。

特に、推薦頻度の高い製品、営業現場でよく使う製品、代替候補として表示される製品から順に、日本語資料を整備します。

### 優先度B: 製品比較機能

Y4825とGPH、5925と5952、467MPと468MP、メタルボンダーとメタルグリップなど、営業現場では「なぜこちらを推奨するのか」を説明する場面が多くあります。

今後は、推奨品と代替候補を横並びで比較できる機能を追加することで、商談時の説明力を高めます。

### 優先度C: 競合品検索・AI相談機能

将来的には、他社品から3M品への置き換え提案や、営業担当者が自由文で相談できるAI相談機能を検討します。

これにより、単なる製品検索ツールではなく、営業提案支援ツールとして発展させることができます。

## 5. 経営向け要約

本ツールは、3M製品選定における営業判断を標準化するための第一版です。

現在、52製品を登録し、営業テスト60件で100%一致を確認済みです。営業担当者が質問に答えるだけで、推奨製品、代替候補、データシートを確認できます。

特に、ベテラン営業の経験に依存しやすい製品の使い分けをロジック化している点が価値です。新人教育、営業品質の平準化、問い合わせ対応のスピード向上に活用できます。

今後は、未登録データシートの整備、製品比較機能、競合品検索、AI相談機能を追加することで、営業現場でより実用的な提案支援ツールへ発展させます。
